¿Qué hicimos?

Creamos un robot que mide el sonido en el aula en tiempo real que emite una señal sonora. Esta señal sirve para que todos notemos que el clima de aprendizaje en el aula se está perdiendo, y podamos bajar el volumen entre todos, sin que el docente tenga que intervenir.
Al mismo tiempo, pensamos en una aplicación que, mediante los datos medidos por nuestro robot, realice graficos que permitan analizar esta información y tomar decisiones informadas para combatir este problema.

Desafío 3: Solución

Para este desafio, debíamos proponer una forma de resolver la problemática elegida, usando a la tecnología como parte de la misma. Así es como nacen nuestros prototipos, una forma de utilizar las herramientas que tenemos para crear una solución.

Robot

Componentes: Sensor KY-038, BUZZER, LEDs, Batería de 3V.

Alt + dB

Alt + dB es un robot diseñado para medir el nivel de ruido en el aula en tiempo real. Cuando detecta un volumen alto y constante, activa una alarma sonora y visual mediante un semáforo de LEDs y un buzzer, alertando a los presentes sin necesidad de intervención docente. Gracias a su microcontrolador Node MCU 1.0 envía automáticamente estos datos a nuestra aplicación para su análisis posterior.

Alt + Ruido

Alt + Ruido es la app que recibe los datos medidos por el robot y permite visualizarlos en gráficos organizados por escuela, turno, aula y hora. Esta herramienta está pensada para docentes y directivos, ya que permite detectar patrones, franjas horarias problemáticas, picos de ruido y tomar decisiones concretas para mejorar el clima de aprendizaje en las aulas.
